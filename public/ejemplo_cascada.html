Presentación en cascada: % de cumplimiento (drill‑down)

-
Cumplimiento ponderado del nodo actual
Leyenda:
  • % de cumplimiento: valor/objetivo (máx. 100%) ponderado por el peso.
  • Si faltan pesos, se distribuyen en partes iguales.
Archivo de ejemplo. Personaliza la variable orgData dentro del código para usar tus datos reales.
